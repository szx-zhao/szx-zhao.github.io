1. 配置

为Github账户设置SSH key

首先检查是否已生成密钥 cd ~/.ssh，ls如果有2个文件，则密钥已经生成，id_rsa.pub就是公钥

如果没有生成，那么通过 命令 来生成

ssh-keygen -t rsa -C "...@qq.com"

生成后，去对应目录C:\Users\Y\ .ssh里（Y为电脑用户名，每个人不同）用记事本打开id_rsa.pub，得到ssh key公钥

切换到github，展开个人头像的小三角，点击settings

然后打开SSH keys菜单， 点击Add SSH key新增密钥，填上标题，跟仓库名保持一致吧，好区分。

接着将id_rsa.pub文件中key粘贴到此，最后Add key生成密钥。到此，github账号的SSH keys配置完成。
